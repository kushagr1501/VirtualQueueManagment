## üì± How It Works

### For Customers
1. Open QueueMaster ‚Üí Select "Customer"
2. Choose a location (clinic, etc.)
3. Select the specific queue you need
4. Receive your position number and verification code
5. Show and verify your code  when called to the required place  - no more confusion!

### For Organizations
1. Open QueueMaster ‚Üí Select "Organization"
2. Create an account or login
3. Access your customized dashboard
4. View and manage all active queues
5. Verify customers using their codes
6. Serve customers efficiently without the chaos

## üõ†Ô∏è Setup Instructions

### Prerequisites
- Node.js (v14+)
- MongoDB
- Git

### Installation

Clone the repository:
```bash
git clone https://github.com/kushagr1501/VirtualQueueManagment.git
cd VirtualQueueManagment
```

### Backend Setup
```bash
cd server
npm install
npm run dev
```

### Frontend Setup
```bash
cd client
npm install
npm run dev
```

### Environment Variables
Create a `.env` file in your backend directory:
```
MONGO_URI=your-mongodb-connection-string
JWT_SECRET=your-secret-key
PORT=5000
```
